# Licensing and Allowed Use

Navigation: [Documentation index](./README.md) · [Official Core plugin](./core-plugin.md) · [Plugin publishing](./plugin-marketplace.md)

Microsoft Rewards Bot is source-available, not open-source under an OSI-style permissive license.

You can read the code, run it for personal noncommercial use, modify it privately, and contribute improvements back to the official project. You cannot redistribute unofficial public copies, sell it, host it as a service, impersonate the official project, or reproduce the proprietary Core plugin.

## Allowed

- personal noncommercial use;
- private local changes;
- forks used to prepare pull requests for the official repository;
- public plugins that use the documented public plugin API;
- small private sharing that keeps the license, notices, attribution, and Core boundary intact.

## Not Allowed

- commercial use without written permission;
- public copied repositories that are not official forks for contribution;
- renamed or rebranded distributions that hide the original project;
- unofficial installers, packages, mirrors, update channels, or hosted services;
- removing notices, attribution, license text, update source references, or Core warnings;
- bypassing, unlocking, replacing, emulating, or reproducing Core;
- publishing code, patches, or instructions designed to avoid a Core license.

## Core Boundary

Core is proprietary. The compiled Core plugin may be run only as supplied by the official project and only with a valid license where required.

Public plugins can extend the bot through the documented plugin API, but they cannot claim official Core status, access private Core APIs, grant premium entitlement, or reproduce Core-only functionality.

## Branding

The project names, logos, banners, badges, screenshots, and visual identity are reserved. Unofficial work must not imply endorsement or official status.

Read [Trademark and branding policy](legal/TRADEMARK.md) for the exact rules.
